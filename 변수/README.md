## 변수 (Variables)
 - 변하는 수, 프로그램 실행시 임시로 저장하는 데이터 장소
 - 관례 : 변수는 선언 및 초기화 후 사용하는 것 권장
  
### 선언 (declaration)
 - var
 - let
 - const : 상수 (constant)
   - 변하지 않는 수
   - const로 선언문을 사용할 경우 초기화 하지 않으면 에러
   - const로 선언한 상수에 값 쓰기 금지

 * 선언만하면 변수는 undefined값

### 초기값 (initialization)
 - 면수 선언시 값을 대입하는 행위
 - let 변수명 = 초기값;
   - 초기값 : 리터럴, 식 (expression)
